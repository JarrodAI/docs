---
title: "GET /api/score"
description: "Retrieve the current T.A.C.O. Score."
---

## Request

```bash
GET https://trumpcheck.one/api/score
```

No parameters required.

---

## Response

```json
{
  "id": "uuid",
  "score": 91,
  "label": "Full TACO",
  "taco_count": 47,
  "holds_count": 5,
  "events_90d": 52,
  "reversal_rate": 0.904,
  "updated_at": "2026-03-04T20:00:00.000Z"
}
```

| Field | Type | Description |
|-------|------|-------------|
| `score` | number | T.A.C.O. Score 0–100 |
| `label` | string | Score label (e.g. "Full TACO", "Moderate", "Holds") |
| `taco_count` | number | Number of documented reversals |
| `holds_count` | number | Number of followed-through policies |
| `events_90d` | number | Total tracked events in 90-day window |
| `reversal_rate` | number | Decimal reversal rate (0–1) |
| `updated_at` | string | ISO timestamp of last update |

---
