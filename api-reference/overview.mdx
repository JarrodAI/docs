## Base URL

All API endpoints are served from:

```
https://trumpcheck.one/api
```

---

## Authentication

The public API endpoints (score, questions, winners) require no authentication.

Admin endpoints require the `CRON_SECRET` header — these are for internal use only and not documented here.

---

## Rate Limiting

| Endpoint | Limit |
|----------|-------|
| `/api/score` | 60 req/min |
| `/api/questions` | 60 req/min |
| `/api/vote` | 5 req/min per IP |
| `/api/winners` | 60 req/min |

---

## Response Format

All responses return JSON. Successful responses include the requested data directly. Errors follow this format:

```json
{
  "error": "Description of what went wrong"
}
```
